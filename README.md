# Personal-Assistant-AI-Agent

An autonomous, local AI agent built with LangGraph and DeepSeek that natively interacts with macOS.

## Features
- **Agentic Orchestration:** Uses LangGraph for state management and reasoning.
- **Deep OS Integration:** Connects directly with macOS Calendar, Apple Notes, Reminders, and system utilities.
- **Web & API Tools:** Features Gmail read/send capabilities, YouTube Music autoplay, Wikipedia, and Tavily Search.
- **Persistent Memory:** Extracts facts and builds a long-term memory of the user.

## Setup
1. Clone the repository.
2. Create a virtual environment and install `requirements.txt`.
3. Copy `.env.example` to `.env` and fill in your API keys.
4. Run `python main.py` or use your terminal alias to chat with the agent!
